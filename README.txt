Por komforte uzi la eventaservo API en via WordPress paĝo

== Description ==

Informu en via WordPress paĝo pri ajna esperantaj eventoj de https://eventaservo.org/.

Vi devas havi uzantkonton en eventaservo.org, por ricevi vian API kodon por funksciigi la aldonaĵon post la instalado.

Viziztu https://eventaservo.org/ por informi vin pri esperantaj eventoj en la tuta mondo!

---

Inform on your WordPress site about any esperanto events from https://eventaservo.org/.

You need an account with eventaservo.org, to get your API Token for the plugin to work.

Visit https://eventaservo.org/ to inform yourself about esperanto events world wide and see how to setup an account.

== Instalado/Installation ==

1. Elpaku la zip-dosierujo al la `/wp-content/plugins/` dosierujo AŬ instalu pere de la aldonaĵejo
2. Aktivigu la aldonaĵo en la 'Plugins' menuo ene WordPress
3. Aldonu vian retpoŝtadreson kaj API kodon ene de la agordejo.
4. Uzu en viaj artikoloj la kodoj por mapo aŭ kalendaro ("evento_kalendaro", "evento_mapo").

1. Unpack the plugin zip into the wordpress `/wp-content/plugins/` directory OR install via plugin search
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your email adress and API token in the admin settings panel of the plugin.
4. Use one of the two shortcodes ("evento_kalendaro", "evento_mapo") in your articles.
