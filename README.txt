Por komforte uzi la eventaservo API en via WordPress paĝo

== Description ==

Informu en via WordPress paĝo pri ajna esperantaj eventoj de https://eventaservo.org/.

Viziztu https://eventaservo.org/ por informi vin pri esperantaj eventoj en la tuta mondo!

---

Inform on your WordPress site about any esperanto events from https://eventaservo.org/.

Visit https://eventaservo.org/ to inform yourself about esperanto events world wide.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-plugin-eventaservo-api.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
